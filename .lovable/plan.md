## План: заменить заказ через Telegram на «Золотой фонд рецептов»

Получил список из таблицы — 40 блюд, сгруппированных по категориям: Супы, Салаты, Плов, Вторые блюда, Выпечка, Десерты, Напитки.

### Что сделать

**1. Удалить Telegram-заказ**
- Удалить файл `src/components/OrderForm.tsx`.
- В `src/pages/Index.tsx` заменить `<OrderForm />` на новый `<Recipes />`.
- В `src/components/Header.tsx` обновить пункт навигации «Заказ» → «Рецепты» (anchor `#recipes`).

**2. Создать `src/components/Recipes.tsx`**
- Заголовок секции: **«Золотой фонд рецептов»** с подзаголовком «Коллекция блюд узбекской кухни».
- Данные — массив объектов `{ category, items[] }` прямо в файле компонента (только названия, без ингредиентов/шагов).
- Вёрстка: группы по категориям, внутри каждой — сетка карточек (Tailwind `grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4`).
- Каждая карточка: иконка из `lucide-react` (UtensilsCrossed / Soup / Salad / Wheat / Cookie / CupSoda по категории), название блюда, лёгкое золотое свечение при hover.
- Использовать существующие классы (`golden-text`, `section-title`, `bg-card`, `animate-fade-in-up`) для единого стиля.
- Stagger-анимация появления карточек через CSS `animation-delay` по индексу.
- Декор: фон-градиент как в текущем `OrderForm`, без блока пара.
